# Suno Lyric Scaffolds — 完整歌词骨架模板
> Version 1.0 | audio-analyzer skill v6 | Updated 2026-03-31
>
> 本文件提供 3 个最常用模板的**完整歌词骨架**，含占位符注释。
> 复制后替换括号内的占位词即可直接提交给 Suno AI。

---

## 📖 占位符说明

| 占位符类型 | 说明 | 示例替换 |
|-----------|------|---------|
| `(主题词)` | 歌曲核心主题名词 | 时光、城市、你、火焰 |
| `(环境词)` | 场景/空间名词 | 街道、深海、夜空、森林 |
| `(情感词)` | 表达情绪的形容词/动词 | 燃烧、破碎、温柔、迷失 |
| `(意象词)` | 具体可视化意象 | 烟火、落叶、星光、镜子 |
| `(时间词)` | 时间/季节 | 深夜、三月、黎明、夏末 |
| `(核心情感)` | 副歌核心词（反复出现）| 自由、痛苦、爱、归途 |
| `(动作词)` | 动词/动作短语 | 奔跑、坠落、飞越、回头 |
| `(场景词)` | 意境场所 | 荒原、云层、记忆里、心门 |
| `(关系词)` | 人际关系动词 | 失去、拥有、相遇、铭记 |
| `(转折句)` | bridge 的情感反转句 | 但原来一切只是幻觉 |
| `(递进句)` | 情感升华描述 | 终于我不再回头 |

## 🔧 快速使用指南

### Step 1 — 从音频分析获取特征
```
BPM: ___
Genre: ___
Has Vocals: yes / no
Danceability: ___
Energy Profile: ___
```

### Step 2 — 选择模板
| 特征组合 | 推荐模板 |
|---------|---------|
| BPM 90-130 + 有人声 + pop/rock | Pop Standard |
| BPM 120+ + 高 danceability + electronic | EDM Build-Drop |
| BPM 60-120 + 无人声 + cinematic | Cinematic/Orchestral |
| BPM 70-115 + hip-hop + 有人声 | Hip-hop / Rap ⬅ scaffold 在本文件 |
| BPM <80 + 有人声 + acoustic | Ballad/Slow ⬅ scaffold 在本文件 |
| BPM <100 + 无人声 + ambient | Minimal/Ambient |
| BPM 100-180 + 高能量 + rock/metal | Rock/Alt-Rock |

### Step 3 — 填入主题词
1. 确定歌曲的**核心情感**（如：孤独、希望、愤怒、爱）
2. 确定**场景/意象**（如：城市夜晚、海边、旧照片）
3. 逐行替换占位符，保持韵脚和音节数

### Step 4 — 提交 Suno
将填好的歌词骨架粘贴到 Suno 的歌词框，结构标签 Suno 会自动识别。

---

> 💡 **Tip**: Suno 对结构标签大小写不敏感，`[Chorus]` 和 `[chorus]` 效果相同。
> 每个段落标签后建议空一行再写歌词内容。
> 如果某段落只需器乐，只保留标签行即可，不填歌词。
